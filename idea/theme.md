# Assets Management System

    - Client Come up With their own S3 Bucket, Google CloudStorage Key with basic Configuration and they can deploy the entire app on servers and can use the app just like the Google Drive.
    - Our application code is identical whether using `AWS` or `GCS`. Switching providers = changing one env variable.

# Features

    - Complete Authentication System with Google Provider and Email/Password Based.
    - User Can Upload Assets of any types with complete support of preview of certain File System.
    - Complete Folder Structured UX in client.
    - Replica of Google Drive

# Techstack

    - `Next.js` For Frontend
    - `Express.js/Node.js` for backend
    - `Typescript` for type safety
    - `Eslint` for strict rules and clean codebase
    - `shadcn/ui` for basic components
    - `pnpm` for package management
    - `zod` for schema validation
    - `React Query` for api integration
